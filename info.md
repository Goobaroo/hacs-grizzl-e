## Grizzl-E EV Charger Integration

Monitor your Grizzl-E EV charger locally in Home Assistant.

### Features
- Real-time charging status and metrics
- Power, current, and voltage monitoring
- Temperature sensors
- Energy and cost tracking
- 30-second polling interval
- No cloud required - works entirely on your local network

### Quick Start
1. Install via HACS
2. Add integration in Home Assistant
3. Enter your charger's IP address (e.g., 192.168.0.2)
4. Enjoy real-time monitoring!

For full documentation, see the [README](https://github.com/goobaroo/hacs-grizzl-e).
